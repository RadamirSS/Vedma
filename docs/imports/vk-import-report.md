## Классификация (семантическая)

Классификация выполнена по смыслу карточки, а не только по ключевым словам в заголовке.

**Товар (физический предмет):** алтарные подставки, покрывала, шопперы, браслеты, серьги, шпильки, бокалы, коробки, коврики, часы, таблички, ритуальные фигурки (Вольт) и др. — даже если в названии есть слова вроде «защита» (например, придверный коврик).

**Услуга (действие / работа):** только позиции, где основное предложение — услуга, а не физический объект:
- Диагностика негатива
- Трансформационная игра «Денежный магнит»

Слова «защита», «помощь», «ритуал» в названии физического товара не переводят карточку в услуги. `needsReview=true` оставлен для 2 позиций с medium confidence после аудита (см. `vk-classification-audit.md`).
